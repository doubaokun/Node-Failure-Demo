Persistance Demo

Overview

This Demo aims to show how Akka can be used to build a resilient and fault-tolerant applicaiton.

This live documentation site provides an access for a cluster of nodes that work together to calculate the GPA for a number of grades. Nodes in the cluster can be eitehr producers or consumers. The producer nodes are responsible for generating the grades while consumer nodes are responslbe for calculating the GPA for that grades.

The code documentation for the platform is avaiable at: Code Doc
